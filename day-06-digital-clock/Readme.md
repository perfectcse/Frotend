⏰ Day 6 – Advanced Digital Clock
📌 Project Overview
This project is a real-time digital clock built using HTML, CSS, and JavaScript. The clock displays the current time, date, and supports switching between 12-hour and 24-hour formats.

✨ Features

Real-time digital clock
Displays current date
AM / PM format
Toggle between 12-hour and 24-hour format
Glow animation UI
Updates every second


🧱 Technologies Used

HTML5
CSS3 (Animations, Glass UI, Glow Effect)
JavaScript (Date Object, setInterval, DOM Manipulation)


🧠 Concepts Learned

JavaScript Date object
setInterval for real-time updates
Time formatting (12h / 24h)
DOM manipulation
Event handling
UI animations
