🎯 Day 10 – Quiz App (JavaScript Logic Project)
📌 Project Overview
This project is a Quiz Application built using HTML, CSS, and JavaScript. The application displays multiple-choice questions, allows users to select answers, tracks the score, shows correct and wrong answers, and displays the final result at the end of the quiz.
This project focuses on JavaScript logic building, arrays of objects, DOM manipulation, and application flow control.

✨ Features

Display quiz questions
Multiple choice answers
Highlight correct answer
Highlight wrong answer
Disable options after selection
Next question button
Score tracking
Progress indicator
Final result display
Restart quiz option
Modern glass UI design


🧱 Technologies Used

HTML5
CSS3 (Glassmorphism, Flexbox, Gradients)
JavaScript (DOM Manipulation, Arrays, Objects, Events, Logic)


⚡ How the Quiz App Works

Quiz questions are stored in an array of objects.
Each object contains:

Question
Answer options
Correct answer index


Question loads dynamically on screen.
User selects an answer.
Correct answer is highlighted.
Score is updated.
User clicks Next to move to next question.
At the end, total score is displayed.
User can restart the quiz.


📊 Quiz Data Structure Example
{
  question: "What does HTML stand for?",
  answers: [
    "Hyper Text Markup Language",
    "High Text Machine Language",
    "Hyperlinks and Text Markup Language",
    "Home Tool Markup Language"
