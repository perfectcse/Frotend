# 🚀 Roadmap Planner

A modern and responsive Roadmap Planner built with **HTML, CSS, and JavaScript**. It helps users create learning roadmaps, manage milestones, track progress, and stay organized. All data is stored locally using **Local Storage**, so your roadmaps remain available even after refreshing the browser.

---



## ✨ Features

- 📚 Create multiple roadmaps
- ➕ Add milestones to each roadmap
- ✏️ Edit roadmap and milestone names
- 🗑️ Delete roadmaps and milestones
- ✅ Mark milestones as completed
- 📊 Automatic progress tracking
- 📈 Dashboard with statistics
- 🔍 Search roadmaps instantly
- 💾 Local Storage support
- 📱 Fully responsive design
- 🎨 Modern UI with animations

---

## 🛠️ Built With

- HTML5
- CSS3
- JavaScript (ES6)

## 🚀 Getting Started

### Clone the Repository

```bash
git clone https://github.com/your-username/roadmap-planner.git
```

### Open the Project

Simply open the `index.html` file in your browser.

No installation or dependencies are required.

---

## 📖 How to Use

1. Enter a roadmap name.
2. Click **Add Roadmap**.
3. Add milestones.
4. Mark milestones as completed.
5. Track your progress through the dashboard.
6. Search roadmaps using the search box.
7. Your data is automatically saved in Local Storage.
