💱 Day 15 – Currency Converter
📌 Project Overview
This project is a Currency Converter web application built using HTML, CSS, and JavaScript.
It allows users to convert currencies in real-time using live exchange rates from an API.
The goal of this project was to learn API integration, currency conversion logic, and better UI handling.

✨ Features

💱 Real-time currency conversion
🌍 Multiple currency support
🔄 Convert between any two currencies
⚡ Live exchange rates using API
🎨 Modern glassmorphism UI
❌ Input validation
📱 Responsive design


🧱 Technologies Used

HTML5
CSS3
JavaScript (ES6+)
Exchange Rate API


⚡ How It Works


App loads all available currencies from API


User enters amount


User selects:

From currency
To currency



JavaScript fetches exchange rates


Currency is converted instantly


Result is displayed dynamically
