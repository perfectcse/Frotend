# 🚀 Day 1 – Profile Card Project (HTML, CSS, JavaScript)

## 📌 What I Built

* A modern Profile Card UI
* Added hover animation
* Follow/Following button toggle
* Clickable social media icons

---

## 🧱 HTML Concepts

* Basic structure (`<!DOCTYPE html>`, `<head>`, `<body>`)
* Semantic tags: `div`, `img`, `h2`, `p`, `button`
* Anchor tag `<a>` for clickable links

---

## 🎨 CSS Concepts

* Flexbox for centering (`display: flex`)
* Gradient background
* Box-shadow for card depth
* Border-radius for smooth UI
* Hover effects using `:hover`
* Transition for smooth animation

---

## ⚡ JavaScript Concepts

* DOM selection (`getElementById`)
* Event handling (`addEventListener`)
* Button toggle logic
* Changing text & styles dynamically

---

## ❌ Mistakes I Learned

* ❌ Cannot use `<link>` inside `<i>`
* ✅ Must use `<a href="">` for clickable links

---

## 💡 Key Learning

* HTML = Structure
* CSS = Design
* JavaScript = Logic

👉 Small projects build strong fundamentals
