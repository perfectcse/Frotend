# 🚀 Day 2 – Animated Navbar (HTML, CSS, JavaScript)

## 📌 Project Overview

This project is a responsive and animated navigation bar built using HTML, CSS, and JavaScript. It includes a mobile-friendly hamburger menu and smooth UI animations.

---

## ✨ Features

* Responsive Navbar (Desktop + Mobile)
* Hamburger menu toggle
* Animated sliding mobile menu
* Active link highlighting
* Hover underline animation
* Icon toggle (menu ↔ close)

---

## 🧱 Technologies Used

* HTML5
* CSS3 (Flexbox, Media Queries, Animations)
* JavaScript (DOM Manipulation, Events)

---

## 📂 Folder Structure

day-02-navbar/
│── index.html
│── style.css
│── script.js

---

## 🧠 Key Learnings

* Responsive design using media queries
* Flexbox layout system
* DOM manipulation using JavaScript
* Event handling and class toggling
* Creating smooth UI animations

---

## 💼 Use Case

This navbar can be used in:

* Portfolio websites
* Landing pages
* Business websites

---

## 🚀 Future Improvements

* Add dropdown menu
* Add sticky navbar on scroll
