🎵 Day 12 – Music Player (Advanced UI)
 Project Overview
This project is a modern Music Player built using HTML, CSS, and JavaScript.
It allows users to play songs, switch tracks, and interact with a clean, animated UI.
This project focuses on audio handling + UI/UX design + interactivity.

✨ Features

Play / Pause music
Next / Previous song
Auto play next song
Progress bar with seek functionality
Current time & duration display
Rotating album cover animation
Dynamic song title & cover image
Smooth UI interactions
Responsive layout


🧱 Technologies Used

HTML5 (Audio API)
CSS3 (Glassmorphism + Animations)
JavaScript (DOM, Events, Logic)


⚡ How It Works


Songs are stored in an array with:

name
audio file
cover image



When user:

Clicks play → audio starts
Clicks next → loads next song
Clicks progress → seeks position



UI updates:

Song title changes
Cover image updates
Progress bar moves
Time updates
