# 🧮 Day 3 – Calculator App (HTML, CSS, JavaScript)

## 📌 Project Overview

This project is a fully functional calculator built using HTML, CSS, and JavaScript. It performs basic arithmetic operations and demonstrates core JavaScript logic and DOM manipulation.

---

## ✨ Features

* Perform basic operations: ➕ Addition, ➖ Subtraction, ✖ Multiplication, ➗ Division
* Real-time input display
* Clear (C) button to reset
* Delete (⌫) button to remove last character
* Equals (=) button to calculate result
* Responsive grid layout
* Smooth button hover and click animations
* Modern glassmorphism UI design

---

## 🧱 Technologies Used

* HTML5
* CSS3 (Flexbox, Grid, Transitions, Glassmorphism)
* JavaScript (DOM Manipulation, Functions, Events)

---

day-03-calculator/

## ⚡ Functionality Implemented

### 🔹 1. Input Handling

* Numbers and operators are appended to the display using JavaScript

### 🔹 2. Clear Function

* Resets the entire display

### 🔹 3. Delete Function

* Removes the last entered character using string slicing

### 🔹 4. Calculation Logic

* Evaluates mathematical expressions using `eval()`

### 🔹 5. Error Handling

* Displays "Error" for invalid expressions using try...catch

---

## 🧠 Key Learnings

* DOM manipulation using `getElementById()`
* Handling user events (`onclick`)
* JavaScript functions and logic building
* String manipulation (`slice()`)
* Error handling using `try...catch`
* Grid layout using CSS

---

## ⚠️ Important Note

* The `eval()` function is used for calculation but is not recommended for production due to security risks.
* In real-world applications, safer alternatives should be used.
