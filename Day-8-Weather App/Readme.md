🌦 Day 8 – Weather App (API Project)
📌 Project Overview
This project is a Weather App built using HTML, CSS, and JavaScript. The application fetches real-time weather data from the OpenWeatherMap API and displays weather information such as temperature, weather condition, humidity, and wind speed for a searched city.
This project introduces working with APIs, asynchronous JavaScript, and handling JSON data.

✨ Features

Search weather by city name
Fetch real-time weather data from API
Display temperature
Display weather description
Display humidity
Display wind speed
Error handling for invalid city
Enter key search support
Modern card UI design


🧱 Technologies Used

HTML5
CSS3 (Flexbox, Gradients, Card UI)
JavaScript (DOM Manipulation, fetch API, async/await)
OpenWeatherMap API


⚡ How It Works

User enters city name
Click Search button or press Enter
JavaScript calls Weather API using fetch()
API returns JSON data
Extract temperature, humidity, wind, weather description
Display data on UI
Show error if city not found


🌐 API Used
OpenWeatherMap API
https://openweathermap.org/api

Building real-world API project
