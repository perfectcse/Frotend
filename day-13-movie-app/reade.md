🎬 Day 13 – Movie Search App (API Project)
📌 Project Overview
This project is a Movie Search Web App built using HTML, CSS, and JavaScript.
It uses the TMDb API to fetch real-time movie data and display it in a modern UI.
The goal of this project was to learn API integration, async JavaScript, and dynamic UI rendering.

✨ Features

🔍 Search movies by name
🎬 Display movie posters
⭐ Show ratings with color indicators
📄 Hover to view movie overview
⏳ Loading state while fetching data
❌ Error handling (API issues / no results)
🎨 Modern responsive UI (Netflix-style)
📱 Clean layout with footer positioning fix


🧱 Technologies Used

HTML5 (Semantic structure)
CSS3 (Grid, Flexbox, Animations)
JavaScript (ES6+)
TMDb API (Real-time movie data)


⚡ How It Works

User enters a movie name in the search bar
JavaScript listens for form submit
API request is sent using fetch()
Data is received in JSON format
Movies are dynamically rendered on the UI
Hover effect shows movie description
