# 🎤 Interview Questions

### 1. What is localStorage?

localStorage stores data in the browser so it remains available even after refreshing the page.

---

### 2. What is CRUD?

CRUD stands for:

- Create
- Read
- Update
- Delete

---

### 3. Why is `filter()` used?

`filter()` returns a new array containing only the items that match a condition.

---

### 4. Why is `find()` used?

`find()` returns the first matching object from an array.

---

### 5. Why is `reduce()` used?

`reduce()` combines all values into a single result, such as calculating total income or total expense.

---

### 6. What is DOM Manipulation?

DOM Manipulation means changing HTML elements dynamically using JavaScript.

---

### 7. Why use localStorage in this project?

It keeps transaction data saved even after the browser is refreshed.
