📝 Day 9 – Notes App (localStorage CRUD Project)
📌 Project Overview
This project is a Notes App built using HTML, CSS, and JavaScript. The application allows users to create, edit, delete, and store notes in the browser using localStorage. Notes remain saved even after refreshing the page.
This project focuses on CRUD operations, DOM manipulation, arrays, and localStorage.

✨ Features

Add new note
Edit existing note
Delete note
Save notes in localStorage
Notes remain after page refresh
Notes counter
Card layout UI
Enter key support to add note
Modern UI design


🧱 Technologies Used

HTML5
CSS3 (Flexbox, Card UI, Shadows)
JavaScript (DOM Manipulation, Events, Arrays, localStorage)

🔹 How It Works

User writes a note
Clicks "Add Note"
Note is stored in an array
Notes are displayed as cards
Notes are saved in localStorage
On page reload, notes are loaded from localStorage
User can edit or delete notes


💾 localStorage Used
The app stores notes using:

localStorage.setItem()
localStorage.getItem()
JSON.stringify()
JSON.parse()

This allows notes to persist after refreshing the page.
